# Load Balancing 
帮助在不断增加的服务器数量上实现横向扩展。

## LB Locations (负载均衡位置)
- **Between user and web server (用户和Web服务器之间)**
- **Between web servers and an internal platform layer (Web服务器和内部平台层之间)** (应用服务器、缓存服务器)
- **Between internal platform layer and database (内部平台层和数据库之间)**

## Algorithms (算法)
- **Least connection (最少连接数)**
- **Least response time (最少响应时间)**
- **Least bandwidth (最少带宽)**
- **Round robin (轮询)**
- **Weighted round robin (加权轮询)**
- **IP hash (IP哈希)**

## Implementation (实现方式)
- **Smart clients (智能客户端)**
- **Hardware load balancers (硬件负载均衡器)**
- **Software load balancers (软件负载均衡器)**
